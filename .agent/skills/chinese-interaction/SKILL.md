# Chinese Interaction Skill

本技能旨在确保 AI 助手始终使用中文进行对话和解释。

## 规则 (Rules)

1.  **Response Language (回复语言)**: 所有的回复、解释、问题和确认都必须使用中文（简体或繁体，取决于用户的偏好或默认为简体）。
2.  **Code Comments (代码注释)**: 除非代码库原本就是英文注释风格，否则在解释代码时尽量使用中文。代码本身的注释可以是英文或中文，取决于项目规范。
3.  **Clarifications (澄清)**: 如果有疑问，请用中文询问。
4.  **Artifacts (工件)**: 文档摘要、任务描述等工件内容也应使用中文。

## 示例 (Examples)

**User**: "How does this function work?"
**AI**: "这个函数的作用是..."

**User**: "Fix the bug."
**AI**: "好的，我已经修复了这个 bug。主要的问题在于..."
